# Sistema POS Moon

Sistema de gestión de punto de venta (POS). Requiere PHP, MySQL y Apache con mod_rewrite.

**Documentación y archivos adicionales:** ver carpeta [extras/](extras/) (documentación, scripts, plantillas de configuración, wiki de usuario).

- Documentación del proyecto: `extras/documentacion/`
- Wiki para usuario final: `extras/wiki/`
- Plantilla de variables de entorno: `extras/config-templates/.env.example`
- Scripts de utilidad: `extras/scripts/`
