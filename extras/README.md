# Extras - Documentación y Archivos Adicionales

Esta carpeta contiene documentación, scripts temporales y archivos que no son parte del sistema principal pero pueden ser útiles para referencia o desarrollo.

## Estructura

```
extras/
├── documentacion/              # Documentación adicional (guías rápidas)
├── documentacion-completa/     # Documentación completa del proyecto
├── scripts/                    # Scripts temporales y de utilidad
├── flujos-n8n/                 # Flujos de trabajo para n8n
├── mejoras/                    # Scripts de migración y mejoras
└── pos-offline-moon/           # Sistema POS offline (módulo separado)
```

## Contenido

### Documentación

- **documentacion/**: Guías rápidas y documentos específicos
- **documentacion-completa/**: Documentación completa del proyecto, incluyendo:
  - Guías de instalación
  - Documentación de integraciones
  - Changelog
  - Scripts de Python para automatización

### Scripts

- Scripts temporales y de utilidad que no son parte del sistema principal

### Flujos n8n

- Configuraciones de workflows para n8n (automatización)

### Mejoras

- Scripts de migración y mejoras que ya fueron aplicadas

### POS Offline

- Sistema POS offline desarrollado en Python (módulo independiente)
