# Scripts de Test y Diagnóstico

Este directorio contiene scripts de prueba y diagnóstico del sistema.

## Scripts de Test

- `test-webhook-accesibilidad.php` - Verifica accesibilidad del webhook de Mercado Pago
- `test-webhook-real.php` - Test con webhooks reales de Mercado Pago
- `test-webhook-qr.php` - Test específico para webhooks de QR

## Scripts de Diagnóstico

- `diagnostico-sistema-cobro.php` - Diagnóstico completo del sistema de cobro
- `diagnostico-webhook.php` - Diagnóstico del webhook de Mercado Pago
- `verificar-webhook-mercadopago.php` - Verificación visual del webhook
- `verificar-webhook.php` - Verificación básica del webhook

## Uso

Estos scripts están diseñados para ser ejecutados desde el navegador o línea de comandos para diagnosticar problemas del sistema.
