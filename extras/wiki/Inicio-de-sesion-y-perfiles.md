# Inicio de sesión y perfiles

En esta página se explica cómo ingresar al sistema y qué puede hacer cada tipo de usuario (perfil).

---

## Cómo ingresar al sistema

1. Abra el sistema en su navegador (la URL le la habrá indicado el administrador o el responsable técnico).
2. En la pantalla de **Inicio de sesión**:
   - **Usuario:** ingrese su nombre de usuario.
   - **Contraseña:** ingrese su contraseña.
3. Haga clic en el botón para iniciar sesión (por ejemplo **Ingresar** o **Entrar**).
4. Si los datos son correctos, se cargará la pantalla principal (Inicio / Tablero). Si no, el sistema mostrará un mensaje de error; verifique usuario y contraseña.

**Importante:** No comparta su usuario y contraseña. Si los olvidó o cree que fueron usados por otra persona, contacte al administrador para que los restablezca.

---

## Cerrar sesión

- En la parte superior de la pantalla suele haber un menú de usuario (nombre o ícono). Desde ahí puede elegir **Cerrar sesión** o **Salir** para cerrar la sesión de forma segura.
- Es recomendable cerrar sesión al terminar su turno, sobre todo si comparte la computadora.

---

## Perfiles de usuario

El sistema distingue dos perfiles: **Administrador** y **Vendedor**. Cada uno ve un menú distinto y tiene permisos diferentes.

### Perfil: Administrador

El **Administrador** puede:

- Ver el **Inicio** con tablero completo (cajas, cuentas corrientes, productos más vendidos, productos recientes).
- Acceder a todo el menú **Empresa**:
  - Datos Empresa
  - Usuarios
  - Listas de Precio
  - Formatos de Balanza
  - Cargar Medios de Pago
- Gestionar **Productos**:
  - Administrar Productos
  - Categorías
  - Combos
  - Imprimir Precios
  - Importar Excel
- Gestionar **Movimientos de productos**:
  - Generar Movimiento
  - Validar Movimiento
  - Movimientos Validados
- Gestionar **Cajas**:
  - Administrar Caja
  - Cierres de caja
- Gestionar **Ventas**:
  - Adm. ventas
  - Adm. presupuestos
  - Crear venta
  - Productos Vendidos
  - Informe rentabilidad
  - Informe de ventas
- Acceder a **Clientes** (y cuenta corriente).
- Gestionar **Compras**:
  - Adm. Compras
  - Crear Compra
  - Ingreso Mercadería
- Acceder a **Proveedores** (y cuenta corriente).
- Acceder a **Integraciones**:
  - Gestionar Integraciones
  - Asistente Virtual

Solo el Administrador puede crear y editar usuarios, listas de precio, formatos de balanza, medios de pago y datos de la empresa.

---

### Perfil: Vendedor

El **Vendedor** tiene un menú reducido, orientado al día a día en caja y ventas:

- **Inicio:** pantalla de bienvenida con accesos rápidos.
- **Administrar Productos:** consultar y, según configuración, editar productos.
- **Imprimir Precios:** imprimir precios para góndola o mostrador.
- **Caja:** acceso a la caja asignada para operar.
- **Adm. ventas:** ver y gestionar ventas (según permisos).
- **Crear venta:** cargar ventas en el punto de venta.
- **Clientes:** consultar y, si corresponde, cargar o editar clientes.
- **Asistente Virtual:** usar el chat/asistente integrado.

El Vendedor **no** ve ni puede usar:

- Datos Empresa, Usuarios, Listas de Precio, Formatos de Balanza, Cargar Medios de Pago.
- Generar/Validar movimientos de productos ni movimientos validados.
- Administrar Caja ni Cierres de caja (sí suele tener “Caja” para operar).
- Presupuestos, informes de rentabilidad o informe de ventas (salvo que se le dé acceso explícito en una versión futura).
- Compras, Proveedores ni Gestionar Integraciones.

En la práctica, el Vendedor se enfoca en: productos, precios, caja, ventas y clientes.

---

## Resumen rápido por perfil

| Función | Administrador | Vendedor |
|--------|----------------|----------|
| Inicio (tablero completo) | Sí | Sí (versión reducida) |
| Empresa, Usuarios, Listas de precio, Formatos balanza, Medios de pago | Sí | No |
| Productos, Categorías, Combos, Imprimir precios, Importar Excel | Sí | Productos e Imprimir precios (según configuración) |
| Movimientos de productos | Sí | No |
| Administrar Caja / Cierres de caja | Sí | No (solo Caja operativa) |
| Ventas, Presupuestos, Crear venta, Informes ventas | Sí | Ventas y Crear venta |
| Clientes | Sí | Sí |
| Compras | Sí | No |
| Proveedores | Sí | No |
| Integraciones (gestionar) / Asistente virtual | Sí | Solo Asistente virtual |

---

## Si no puede ingresar o le falta algo en el menú

- **No puede ingresar:** compruebe usuario y contraseña; si sigue fallando, pida al administrador que revise que su usuario esté activo y con el perfil correcto.
- **Le falta una opción en el menú:** su perfil (Vendedor/Administrador) define qué ve. Si cree que debería tener acceso a algo, contacte al administrador para que revise su perfil y permisos.

Para crear o modificar usuarios y perfiles, el administrador debe usar **Empresa → Usuarios**. Ver [Empresa](Empresa).
