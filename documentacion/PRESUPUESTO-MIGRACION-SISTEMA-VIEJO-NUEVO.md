## 1. Descripción del Servicio

Se propone la **migración completa** del sistema de gestión comercial (POS) que actualmente utiliza el cliente hacia la nueva versión del software, manteniendo toda la información histórica (ventas, clientes, productos, proveedores, etc.) y adaptando la operación para que el personal pueda seguir trabajando con normalidad desde el primer día.

El trabajo incluye el análisis del sistema actual, la preparación de los datos, la instalación del nuevo sistema, la configuración de todos los parámetros de negocio y la adaptación del módulo de facturación electrónica que el cliente ya utiliza, de modo que quede integrado y fácil de usar para los operadores.

---

## 2. Trabajos Incluidos

### 2.1 Análisis y preparación
- Revisión del sistema actual y de la base de datos
- Respaldo completo de datos y archivos antes de cualquier cambio
- Identificación de diferencias entre la estructura actual y la nueva
- Generación de los scripts necesarios para actualizar la base de datos sin pérdida de información

### 2.2 Migración de datos
- Traspaso de ventas históricas al nuevo formato (de detalle en texto a tabla relacional)
- Migración de productos, categorías, clientes, proveedores y movimientos
- Corrección de datos duplicados o inconsistentes
- Carga de listas de precios, medios de pago, formatos de balanza y permisos según el uso actual

### 2.3 Instalación del nuevo sistema
- Instalación limpia del nuevo software en el servidor del cliente
- Configuración del entorno (versión de PHP, base de datos, etc.)
- Restauración de certificados de facturación electrónica (AFIP)
- Configuración de sucursales, puntos de venta y almacenes

### 2.4 Adaptación del módulo de facturación
- Integración del sistema de facturación electrónica que el cliente ya utiliza
- Configuración de tipos de comprobante, condiciones de IVA y datos fiscales
- Ajuste de la interfaz para que sea sencilla para el usuario final (cajeros, vendedores)
- Pruebas de emisión de facturas, notas de crédito y comprobantes asociados

### 2.5 Configuración de usuarios y permisos
- Creación o actualización de usuarios con sus respectivos permisos
- Asignación de sucursales, puntos de venta y listas de precio por usuario
- Configuración de roles (Administrador, Vendedor, etc.)

### 2.6 Validaciones y ajustes finales
- Verificación de que no existan ventas, clientes o productos duplicados
- Aplicación de controles para evitar duplicados en futuras operaciones
- Pruebas generales de ventas, compras, reportes y facturación

### 2.7 Módulo de retenciones ARCA
- Instalación y configuración del módulo de retenciones ARCA
- Integración con el flujo de facturación y compras del sistema

### 2.8 Soporte durante la migración
- Acompañamiento durante el proceso de migración
- Resolución de inconvenientes que surjan en la instalación o configuración
- Documentación básica de los cambios realizados
